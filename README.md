# dnet-tui

A terminal user interface (TUI) application built with Rust.

## Installation

### Using Cargo

```sh
cargo install  https://github.com/firstbatchxyz/dnet-tui.git
```

### From Source

```sh
git clone https://github.com/firstbatchxyz/dnet-tui.git
cd dnet-tui
cargo build --release
```

## Usage

Run the application:

```sh
dnet-tui
```
